* **`api/`** : Contient tous les appels API et la logique de communication avec le serveur (requêtes fetch vers les contrôleurs PHP).
* **`components/`** : Regroupe les composants d'interface réutilisables à travers l'application (page d'accueil, Footer, Sidebar de navigation, etc.).
* **`pages/`** : Contient les différentes vues principales liées au système de routage (Espace Admin, Gestion des Bénévoles, Gestion des Missions, Formulaires de connexion et d'inscription, etc.).
